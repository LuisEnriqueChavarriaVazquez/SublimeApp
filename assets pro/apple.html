Sublimetricks.

Apple.


  • Comandos generales.

    + + p
    Acceso a la paleta de comandos.

    + K, + B
    Mostrar la sidebar con archivos ocupados.

  • Edición.

    + x
    Cortar una linea de codigo.

    + ENTER
    Insertarse una linea después.

    + L
    Selecciona una linea y si se repita continua seleccionando lineas hacia abajo.

    + D
    Elige una palabra y continua Ctrl + D para elegir repeticiones de la palabra.

    + + k
    Elemina una linea de codigo.

    + ]
    Genera un comentario en la linea actual.

    + [
    Elimina el comentario racien generado.

    + + d
    Duplica la linea actual.

    + j
    Junta la linea de abajo al final de la linea actual.

    + a
    Seleccionar todo.

    + y
    Repita una accion anterior.

    + + v
    Pega una sección copiada.

    + ESPACIO
    Autocompleta sugerencias de Sublime Text.

  • Navegación.

    + P, + T
    Despliega menu rapido con archivos por nombre.

    + R
    Nos permite acceder a todos nuestros atributos en las etiquetas.

  • Tabs.

    + n
    Nueva pestaña.

    + + n
    Abre una nueva ventana.

    + + t
    Abre la ultima ventana abierta

  • Busqueda de elementos.

    + f
    Te muestra un buscador de elementos.

    + + f
    Inicia busqueda en tus archivos.

  • Elementos de la ventana.

    + alt + 1
    Te muestra solos una columna en caso de que tengas varias.

    + alt + 2
    Te muestra dos columnas en todo el documento.

    + alt + 3
    Te muestra 3 columnas en todo el archivo.

    + alt + 4
    Te muestra 4 columnas en todo el archivo.

    + alt + 5
    Te divide la ventana en 4 grupos pero no como columnas.

  • Bookmarks.

    + F2
    Agregar y poner un Bookmark (Un separador).

    F2
    Viajar entre bookmarks.

    + F2
    Viajar al bookmark anterior.

  • Textos.

    + KU
    Transforma el texto a mayusculas.

    + KL
    Transforma el texto a minusculas.
